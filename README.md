# CS380 Term Project

## Overview
This project is a web-based online store designed for Jacksonville University. The website allows students to browse and purchase a variety of items, including campus merchandise, stationery, and tech products. The goal of this project is to provide a simple, user-friendly shopping experience while demonstrating core web development skills using HTML, CSS, and JavaScript.

## Features
- Browse products by category (Merch, Stationery, Tech)
- View product details (name, price, description, image)
- Add items to a shopping cart
- Responsive design for desktop and mobile users
- Clean and modern user interface

## Technologies Used
- HTML5: Structure of the website
- CSS: Styling and Input
- JavaScript: Interactivity and dynamic features
- Git & GitHub: Version control & deployment
- GitHub Pages: Hosting a website
